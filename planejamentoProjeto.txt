1- ler arquivo 
2- pegar cada palavra e guardar em um array
2- Contar cada vez que uma nova palavra é adicionada, para saber o tamanho do array
contador:

 lw $t1, iterator

 addi $t1, $t1, 1      #a cada novo incremento, ele soma +1

3- cada palavra lida ficará em uma posicao do array
4- sortea uma posição 
5- multiplica por 4 o número sorteado para conseguir acessar qualquer posição do array
obs: 0 - 4 - 8 - 12 - .... (de 4 em 4 bits)
6- verifica se existe essa posição no array (se não é maior que o array)
   compara posição com o tamanho do array feito pelo contador
7- se for dentro do tamanho do array, acessa a posição e guarda a palavra selecionada
8- Fazer Input do usuário, para ele poder entrar com as letras e brincar
